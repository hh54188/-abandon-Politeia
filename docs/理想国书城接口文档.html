图书 ¶

1. 分类图书列表 ¶

获取指定分类下的图书列表，支持按照分页加载。

  • 请求URL

api/ebook/books

  • 请求方式

GET

  • 请求参数
请求参数 参数类型 参数说明
cate_id Integer,必填 分类ID
start Integer,可为空 起始offset数，默认0
count Integer,可为空 每页结果条数，默认10，最大50
  • 返回参数
返回参数 参数类型 参数说明
code Integer 执行结果code，成功200
msg String 执行结果消息
data Array 执行结果消息

2. 搜索图书列表 ¶

获取指定关键词图书列表，包括全部分类、指定分类、个人书架（阅读历史、已购买、已借阅）。 支持按照分页加载。

  • 请求URL

api/ebook/search

  • 请求方式

GET

  • 请求参数
请求参数 参数类型 参数说明
q String,必填 关键词
cate_id Integer,可为空 指定的分类ID
shelf Integer,可为空 只搜索个人书架时，需指定shelf=1，默认0
start Integer,可为空 起始offset数，默认0
count Integer,可为空 每页结果条数，默认10，最大50
  • 返回参数
返回参数 参数类型 参数说明
code Integer 执行结果code，成功200
msg String 执行结果消息
data Array 执行结果消息
  • 返回示例
{
  "code": 200,
  "msg": "ok",
  "data":{
    "start": 0,  //传参start
    "count": 10,  //每页结果条数
    "total": 30,  //结果总数
    "list":[
    {
      "id": 10001,  //图书ID
      "name": "看见",  //图书名
      "author_name": "柴静",  //作者名
      "desc": "柴静十年央视历程，中国十年社会变迁备忘录",  //图书内容摘要
      "cate_id":  1, //分类ID

    {
      "id": 10002,  //图书ID

2. 发评论 ¶

为某本图书打分、发表评论。

  • 请求URL

api/ebook/add_review

  • 请求方式

POST

  • 请求参数
请求参数 参数类型 参数说明
book_id Integer,必填 图书ID
rate_num Integer,必填 评分值，1-10分
content String,必填 评论内容
  • 返回参数
返回参数 参数类型 参数说明
code Integer 执行结果code，成功200
msg String 执行结果消息

  "code": 200,
  "msg": "ok",
}

  "code": 101,

阅读 ¶

1. 图书正文内容 ¶

获取图书指定章节（或指定进度）的正文内容，每次返回一个章节的完整内容。指定进度的获取正文的，前台阅读页面根据返回的offset进行分页及定位处理。 未购买、借阅的图书，只能阅读免费试读的部分。

  • 请求URL

api/ebook/content

  • 请求方式

GET

  • 请求参数
请求参数 参数类型 参数说明
book_id Integer，必填 图书ID
chapter_id Integer，chapter_id 和 progress必填一项 章节ID
progress Integer，chapter_id 和 progress必填一项 内容进度，825表示825‰，不传千分比符号。

返回参数 参数类型 参数说明
code Integer 执行结果code，成功200
msg String 执行结果消息
data Array 执行结果消息

获取指定图书的目录章节列表。章节目录最多有2级。

  • 请求URL

api/ebook/toc

  • 请求方式

GET

  • 请求参数
请求参数 参数类型 参数说明
book_id Integer,必填 图书ID
  • 返回参数
返回参数 参数类型 参数说明
code Integer 执行结果code，成功200
msg String 执行结果消息
data Array 执行结果消息
  • 返回示例
{
  "code": 200,
  "msg": "ok",
  "data":{
    "total": 12,  //章节总数
    "list":[
    {
      "id": 8812,  //章节ID
      "name": "第二章 知识分子与他的时代",  //章节标题
      "start_offset": 92,  //本章起始的进度值，千分数
      "end_offset": 172, //本章结束的进度值，千分数
      "word_count": 13781, //字数
      "is_free": 1, //是否免费试读，1为免费试读的
      "chapters": [ //二级章节。大部分图书只有一级，chapters字段返回[]
          {
               "id": 8813,  //章节ID
